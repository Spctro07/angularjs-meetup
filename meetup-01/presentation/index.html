¿Que es AngularJS?

  • Framework JavaScript Open Source
  • Aplica el patrón MVC
  • Es desarrollado por Google

Principales ventajas

  • Aplica Dependency Injection
  • Facilita la construcción de SPA con el uso de rutas y bindings
  • Permite desacoplar la manipulación del DOM de la lógica
  • Extiende el vocabulario HTML
  • Buenas prácticas de Inyección de Dependencias
  • Facilidad de realizar Testing Unitario y E2E
  • Los objetos JS no necesitan una estructura específica para Getters y Setters

Controller con Repeater


<div ng-controller="ItemsCtrl">
    <button ng-click="addItem()">Add Item</button>
    <table>
        <tr ng-repeat="item in items">
            <td>{{item}}</td>
        </tr>
    </table>
</div>
                    

<script>
    function ItemsCtrl($scope) {
        $scope.count = 0;
        $scope.items = [];
        $scope.addItem = function () {
            $scope.count++;
            $scope.items.push('Nuevo Item ' + $scope.count);
        }
    }
</script>
{{item}}
El Scope es representa el contexto de ejecución.

Controller

  • Utiliza un $scope para comunicar datos entre el Controller y la Vista
  • El Modelo está dentro del $scope

function TaskCtrl($scope){
	$scope.tasks = [ 
	    { title: 'Fix Bug #3457', status: 'Done' }, 
	    { title: 'Add new Feature', status: 'Pending' } 	
	]
	$scope.addTask = function (task) {
		$scope.tasks.push(task);
	}
}

    • Filtrar una lista
